# Декодирование сообщений
## Описание
Данная программа считывает файл с сообщениями Mode S, содержащими временные метки и шестнадцатеричные данные.

---
## Описание работы
Данная программа извлекает:
* Временные метки — количество секунд с начала эпохи UNIX.
* ICAO — уникальный адрес самолета.
* Координаты — широту и долготу для построения трека.
* Курс — направление движения борта в градусах.

После обработки файла программа выводит в консоль таблицу ICAO-адресов с датой и временем (зоны UTC и МСК), когда борт впервые и в последний раз был зафиксирован, а также показывает, передавал ли он данные о координатах и курсе.

Затем программа строит интерактивные графики:

* Схема трека полета (долгота vs широта).
* Изменение координат по времени.
* Изменение курса по времени.

## Как использовать
1. Убедиться, что установлен Python 3.
2. Установить менеджер пакетов ```pip``` (если не установлен):
```sudo apt-get install python3-pip```
3. Затем установить библиотеки numpy, pyModeS, pytz и import matplotlib:
```pip3 install numpy```
```pip3 install pyModeS```
```pip3 install matplotlib```
```pip3 install pytz```
4. Запустить в терминале программу:
```python3 prog1.py```
5. Если в программе не прописан файл или он находится в другом месте можно указать к нему путь (флаг -f):
```python3 prog1.py -f 2025-10-03.1759515715.074510429.t4433```
6. Для анализа сообщений для конкретного борта можно использовать флаг -a:
```python3 prog1.py -f 2025-10-03.1759515715.074510429.t4433 -a 4248E2```
